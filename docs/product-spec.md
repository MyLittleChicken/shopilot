# Shopilot 제품 명세

자연어로 말하면 상품을 탐색·비교하고 구매 판단까지 돕는 AI 커머스 에이전트. 단순 챗봇이 아니라, 사용자의 문제를 정의하고 에이전트의 판단 흐름을 거쳐 실제 구매 경험으로 연결하는 것을 목표로 한다.

아키텍처와 기술 스택은 `.claude/architecture.md`, 진행 상황은 `docs/PROGRESS.md`를 참조한다.

## 주요 기능

### 1. AI 에이전트 커머스
자연어 입력을 바탕으로 상품을 탐색·추천·비교하고 구매 판단까지 돕는다. LLM과 에이전트 구조(Tool/Function Calling, RAG)로 실제 사용자 문제를 해결한다.

### 2. 실시간 채팅
사용자와 에이전트가 자연스럽게 대화하는 채팅 인터페이스. SSE/스트리밍 기반 응답, 타이핑·로딩 표시, 대화 상태 동기화를 제공한다.

### 3. 자체 커머스 경험 (웹)
개별 쇼핑몰에 종속되지 않는 Shopilot 자체 B2C 웹. 사용자가 상황·취향·목적을 말하면 에이전트가 여러 상품을 탐색·추천하는 쇼핑 경험을 만든다.

### 4. 임베더블 위젯
외부 쇼핑몰에 설치되는 채팅 위젯. 삽입용 JS 번들, 위젯 UI, 초기화 스크립트, CDN 배포, Shadow DOM 격리를 갖춘다.

### 5. LLM 응답 품질
응답이 실서비스에서 안정적으로 동작하도록 검증한다. 프롬프트 테스트, 응답 포맷 검증(Zod), 실패 응답 폴백·재시도, 추천 품질 평가, 이상 입력 대응.
